## Установка
### 1. Подготовка
1. Наличие архива сборки
2. Наличие лаунчера, поддерживающего импорт zip инстанса [Prism](https://prismlauncher.ru) | [MultiMC](https://multimc.org) | Любой бесплатый

### 2. Установка сборки.
1. Закидываем инстанс в лаунчер или добавить сборку через "Импорт"
2. Ожидание установки и подкачки файлов.Можеь понадобится доступ к NeoForge. 
3. Запускаем.
